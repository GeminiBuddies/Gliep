# Symbol Allocation Table

- `!`: local variable definition
  - `!!`: global variable reference
- `"`: string
- `#`: comment
- `$`: function call
- `%`: modulus
- `&`: and
  - `&&`: short-circuit and
- `(` `)`: block
- `*`: multiplication
- `+`: addition
- `,`: separator in lists and table literals
- `-`: subtraction
  - `->`: right arrow in lambdas and function definitions
- `.`: index by name
  - `.!`: index locally by name
- `/`: division
- `:`: separator between key and value in table literals
- `;`: statements separator
- `<`: less than
  - `<<`: left shift
  - `<=`: less than or equal to
  - `<|`: pseudo-index head
- `>`: greater than
  - `>>`: right shift
  - `>=`: greater than or equal to
- `=`: assign
  - `==`: equal to
- `?`: is nil
  - `??`: nil coalescing
- `@`: index by variable
  - `@!`: index locally by variable
- `[` `]`: on stack list
- `^`: xor
- `` ` ``: unary op prefix
- `{` `}`: table literal
- `|`: or
  - `|>`: pseudo-index tail
  - `||`: short-circuit or
- `~`: not
  - `~=`: not equal to

## Not used yet

- `\`
- `'`
